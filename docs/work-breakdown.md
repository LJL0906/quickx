# 需求拆分 (WBS)

> 版本: v0.1 | 日期: 2025-07

---

## 拆分说明

每个功能点按 **P0 / P1 / P2** 标注优先级：
- **P0** — MVP 必须，缺了就不算可用产品
- **P1** — 重要但可延后，首版可降级为简化版
- **P2** — 锦上添花，首版不做

---
