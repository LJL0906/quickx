# UI 设计文档

> 版本: v0.1 | 日期: 2025-07

---

## 1. 设计原则

- **极简** — 搜索浮窗不显多余元素，纯功能导向
- **快速** — 操作路径最短，核心动线 ≤ 2 步
- **低调** — 不打扰用户主工作流，配色收敛
- **一致** — 两个窗口共享设计语言

## 3. 搜索浮窗 (Search Bar)

### 3.1 尺寸和位置

| 属性 | 值 |
|------|-----|
| 宽度 | 600px |
| 高度 | 收起 56px / 展开（有结果）自适应，最大 420px |
| 圆角 | 12px |
| 位置 | 屏幕水平垂直居中 |
| 阴影 | `0 8px 32px rgba(0,0,0,0.12)` |

### 3.2 状态一：初始态（无输入）

```
                  ┌───────────────────────────────────────────┐
                  │                                           │
                  │  🔍   搜索链接...                     📋   │
                  │                                           │
                  └───────────────────────────────────────────┘
                         ↑ 输入框  ↑ placeholder  ↑ 详情入口
```

- 焦点自动在输入框
- placeholder 灰色显示「搜索链接...」
- 📋 图标 hover 时变色（#6C8CFF），tooltip「打开详情窗口」

### 3.3 状态二：输入中 + 有结果

```
                  ┌───────────────────────────────────────────┐
                  │                                           │
                  │  🔍   react│                          📋   │
                  │                                           │
                  ├───────────────────────────────────────────┤
                  │                                           │
                  │  📎  React 官方文档                        │
                  │      react.dev                            │
                  │  ───────────────────────────────────────  │
                  │  📎  React GitHub                         │
                  │      github.com/facebook/react            │
                  │  ───────────────────────────────────────  │
                  │  📎  React Router 文档                     │
                  │      reactrouter.com                      │
                  │                                           │
                  └───────────────────────────────────────────┘
                     ↑ 当前选中项高亮（#2A2A3C 背景 + 左侧蓝色指示条）
```

- 输入框内显示输入内容，光标闪烁
- 下拉列表在输入框下方，与输入框无缝衔接
- 每条结果：图标 + 标题（白色） + URL（灰色小字）
- 当前选中行：背景色变为 #F0F0F6，左侧 3px 蓝色竖线（#4A6CF7）
- 超出 6 条时内部滚动

### 3.4 状态三：无匹配结果

```
                  ┌───────────────────────────────────────────┐
                  │                                           │
                  │  🔍   xxxxxx│                         📋   │
                  │                                           │
                  ├───────────────────────────────────────────┤
                  │                                           │
                  │           😕 没有匹配的链接                 │
                  │                                           │
                  └───────────────────────────────────────────┘
```

### 3.5 动画

| 动画 | 说明 |
|------|------|
| 出现 | 150ms ease-out，scale(0.95→1) + opacity(0→1) |
| 消失 | 100ms ease-in，opacity(1→0) |
| 下拉展开 | 200ms ease-out，max-height 过渡 |

## 4. 详情窗口 (Main Window)

### 4.1 尺寸和位置

| 属性 | 值 |

| 默认高度 | 600px |
| 最小尺寸 | 600×400 |
| 标题栏 | 系统原生 |
| 圆角 | 无（标准窗口） |

### 4.2 全局布局

```
┌──────────────────────────────────────────────────────────┐
│  QuickX                                    ─   □   ✕    │  ← 标题栏
├──────────────────────────────────────────────────────────┤
│  [🔗 链接] [📝 笔记] [💻 代码] [📋 剪切板] [⚙ 设置]       │  ← Tab 栏 48px
├──────────────────────────────────────────────────────────┤
│                                                          │
│                                                          │
│                    内容区域                               │
│                                                          │
│                                                          │
│                                                          │
├──────────────────────────────────────────────────────────┤
│  [+ 新建链接]                             共 12 条       │  ← 底部栏 40px
└──────────────────────────────────────────────────────────┘
```

### 4.3 Tab 栏

```
┌──────────────────────────────────────────────────────────┐
│  ┌──────────┐ ┌──────────┐ ┌──────────┐ ┌──────────┐ ... │
│  │ 🔗 链接   │ │ 📝 笔记  │ │ 💻 代码  │ │ 📋 剪切板 │    │
│  │ ● 12     │ │   8      │ │   15     │ │   200    │    │
│  └──────────┘ └──────────┘ └──────────┘ └──────────┘    │
│    ↑ 活跃态     ↑ 非活跃态                                │
└──────────────────────────────────────────────────────────┘
```

- 活跃 Tab：底部蓝色下划线 3px + 文字白色 + 背景微亮
- 非活跃 Tab：文字灰色 + 无下划线
- 每个 Tab 下方显示条目数量
- hover 非活跃 Tab：文字变亮

## 8. 交互细节汇总

| 交互 | 触发 | 反馈 |
|------|------|------|
| 唤起浮窗 | `Alt+Q` | 浮窗 150ms 渐显 + 输入框聚焦 |
| 搜索 | 键盘输入 | 150ms 防抖后下拉结果 |
| 打开链接 | `Enter` | 浏览器打开 + 浮窗消失 + open_count +1 |
| 打开详情 | 点 📋 | 浮窗消失 + 详情窗口显示 |
| 关闭详情 | 点 ✕ | 窗口隐藏到托盘 |
| 复制剪切板 | 点击条目 | clipboard.writeText + Toast |
| 复制代码 | 点 📋 按钮 | clipboard.writeText + Toast |
| 退出应用 | 托盘右键 → 退出 | 停止监听 → 关库 → 退出 |
