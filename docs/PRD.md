## 1. 产品定义

### 1.1 产品名称

**QuickX**（待定）

### 1.2 产品定位

一款 Windows 桌面效率工具，通过 `Alt+Q` 唤出**搜索浮窗**，核心场景是快速搜索已保存的链接并一键打开浏览器。同时提供笔记、代码片段、剪切板历史等模块，在独立的**详情窗口**中管理。常驻系统托盘，支持开机自启动。

### 1.3 目标用户

- 开发者、效率型用户
- 收藏了大量技术文档/工具链接、需要快速检索打开的人
- 需要轻量笔记和代码片段管理的用户

### 1.4 核心体验

```
┌─────────────────────────────────────────────────────────┐
│                      两层窗口模型                         │
│                                                         │
│   Alt+Q ──→ ┌──────────────────┐                        │
│             │  🔍 搜索链接...  │  ← 搜索浮窗（Search Bar）│
│             │           [📋]   │    极简、Alfred 风格     │
│             └──────────────────┘                        │
│                  │ 回车 ──→ 浏览器打开链接                │
│                  │ 点 📋 ──→ 打开详情窗口                 │
│                  ↓                                      │
│             ┌──────────────────────────┐                │
│             │  笔记 │ 代码 │ 剪切板 │⚙ │  ← 详情窗口      │
│             │                        │    完整管理后台   │
│             │   (完整功能区域)         │                │
│             └──────────────────────────┘                │
└─────────────────────────────────────────────────────────┘
```

**核心理念：搜索浮窗随叫随到、用完即走；详情窗口完整管理、按需打开。**

## 2. 窗口体系

### 2.1 搜索浮窗（Search Bar）— 第一层

| 项目 | 说明 |
|------|------|
| 唤起方式 | `Alt+Q` 全局快捷键 |
| 隐藏方式 | `Esc` / 点击浮窗外部 / 再次按 `Alt+Q` |
| 窗口形态 | 极简搜索栏，约 600×50px，居中偏上 |
| 外观 | 圆角、阴影、置顶（always-on-top）、无边框 |
| 组成 | 搜索图标 + 输入框 + 右侧详情入口图标 |
| 输入框 placeholder | "搜索链接..." |

**浮窗线框图：**

```
┌─────────────────────────────────────────────┐
│  🔍  │  输入关键词搜索链接...        │  📋   │
└─────────────────────────────────────────────┘
         ← 输入框（实时匹配）→           ↑
                                    点击进入详情窗口
```

**交互逻辑：**

| 操作 | 行为 |
|------|------|
| 输入关键词 | 实时模糊搜索已保存的链接（匹配标题和 URL） |
| `↑↓` 键 | 在搜索结果列表中导航 |
| `Enter` | 用系统默认浏览器打开当前选中链接 |
| 点击 📋 图标 | 隐藏搜索浮窗，打开详情窗口（主窗口） |
| `Esc` | 隐藏浮窗 |
| 失焦 | 隐藏浮窗 |

**搜索结果下拉列表：**

```
┌─────────────────────────────────────────────┐
│  🔍  │  react                             │  📋   │
├─────────────────────────────────────────────┤
│  📎 React 官方文档                           │
│     https://react.dev                       │
│  ─────────────────────────────────────────  │
│  📎 React GitHub 仓库                        │
│     https://github.com/facebook/react       │
│  ─────────────────────────────────────────  │
│  📎 React 中文文档                           │
│     https://zh-hans.react.dev               │
└─────────────────────────────────────────────┘
```

> 搜索结果列表在输入框下方展开，最多显示 6 条，超出滚动。

### 2.2 详情窗口（Main Window）— 第二层

| 项目 | 说明 |
|------|------|
| 打开方式 | 搜索浮窗 📋 图标 / 托盘右键菜单「打开主窗口」 |
| 窗口形态 | 标准桌面窗口，约 800×600px，可拖拽、调整大小 |
| 外观 | 标准标题栏、最小化/最大化/关闭按钮 |
| 关闭行为 | 关闭窗口 = 隐藏到托盘（不退出应用） |
| 内容 | Tab 切换：链接管理 / 笔记 / 代码片段 / 剪切板 / 设置 |

**详情窗口线框图：**

```

├──────────────────────────────────────────────────┤
│                                                  │
│   当前模块的内容区域                               │
│                                                  │

└──────────────────────────────────────────────────┘
```

## 3. 功能模块

### 3.1 链接管理 🔗

用户在详情窗口中管理所有保存的链接，这些链接也是搜索浮窗的数据源。

**展示方式：药丸布局（Pill Layout）**

链接以圆角胶囊/药丸形卡片排列，每个卡片显示名称、网址和备注，点击直接打开浏览器。

```
┌──────────────────────────────────────────────────┐
│  [+ 新增链接]                          🔍 搜索... │
├──────────────────────────────────────────────────┤
│                                                  │
│  ┌─────────────────────┐  ┌─────────────────────┐│
│  │ 🏠 React 官方文档    │  │ 📦 Vue 3 中文文档    ││
│  │ react.dev           │  │ cn.vuejs.org        ││
│  │ React 入门必看       │  │ 官方中文教程         ││
│  └─────────────────────┘  └─────────────────────┘│
│                                                  │
│  ┌─────────────────────┐  ┌─────────────────────┐│
│  │ 🔧 Rust 语言圣经     │  │ 📘 TypeScript 手册   ││
│  │ course.rs           │  │ typescriptlang.org  ││
│  │ 最好的 Rust 中文教程  │  │ 官方手册，随时查阅    ││
│  └─────────────────────┘  └─────────────────────┘│
│                                                  │
└──────────────────────────────────────────────────┘
```

> 卡片自动换行排列（CSS flex-wrap），每行放 3-4 个，随窗口宽度自适应。

**功能：**

| 功能 | 说明 |
|------|------|
| 新增链接 | 名称 + 网址 + 备注 + 标签，网址必填 |
| 编辑/删除 | 支持修改名称、网址、备注、标签 |
| 分类筛选 | 按标签过滤展示 |
| 搜索 | 顶栏搜索框，按名称、网址、备注模糊搜索 |
| 打开链接 | 点击药丸卡片 → 系统默认浏览器打开（open_count +1） |
| 右键菜单 | 卡片右键 → 编辑 / 删除 / 复制网址 |
| 排序 | 按创建时间倒序 / 按打开次数 |

**字段：**

| 字段 | 说明 |
|------|------|
| title | 链接名称 |
| url | 链接地址 |
| remark | 备注说明 |
| tags | 标签（JSON 数组） |
| favicon | 网站图标 URL（可选，自动获取） |
| open_count | 打开次数 |
| created_at | 创建时间 |
| updated_at | 更新时间 |

## 4. 系统行为

### 4.1 生命周期

```
系统启动（若开启自启动）
  └→ 主进程启动
       ├→ 初始化 SQLite
       ├→ 注册全局快捷键 Alt+Q
       ├→ 启动剪切板监听（若开启）
       └→ 显示托盘图标

用户交互
  ├→ Alt+Q ──→ 唤起搜索浮窗
  │    ├→ 搜索链接 → Enter → 打开浏览器
  │    └→ 点击 📋 → 打开详情窗口
  ├→ 托盘左键 ──→ 唤起搜索浮窗
  └→ 托盘右键 ──→ 打开详情窗口 / 设置 / 退出

用户退出
  └→ 托盘右键 → 退出
       └→ 停止剪切板监听 → 关闭数据库 → 退出进程
```

### 4.2 窗口关系

```
搜索浮窗 (alwaysOnTop, 无边框, 轻量)
  │
  ├─→ Esc / 失焦 → 隐藏（不关闭）
  ├─→ Enter 链接 → 打开浏览器（浮窗自动隐藏）
  └─→ 点击 📋 → 隐藏浮窗 + 打开详情窗口

详情窗口 (标准窗口, 可最小化)
  │
  ├─→ 关闭按钮 → 隐藏到托盘（不退出）
  └─→ 托盘右键 → 退出 → 整个应用退出
```

### 4.3 开机自启动

- Electron `app.setLoginItemSettings()` 实现
- 设置页提供开关

### 4.4 托盘

| 操作 | 行为 |
|------|------|
| 左键单击 | 唤起搜索浮窗 |
| 右键菜单 | 打开详情窗口 / 关于 / 退出 |

## 5. 技术方案

### 5.1 技术栈

| 层 | 技术 | 说明 |
|----|------|------|
| 框架 | Electron 28+ | 跨平台桌面框架 |
| 主进程 | TypeScript | 窗口管理、快捷键、托盘、剪切板、IPC |
| 渲染进程 | React 18 + TypeScript | 两个 SPA：搜索浮窗 + 详情窗口 |
| 构建工具 | Vite | 渲染进程打包 |
| 数据库 | SQLite (better-sqlite3) | 同步 API，适合 Electron |
| 打包 | electron-builder | 生成 exe / NSIS 安装包 |
| Markdown | marked | 笔记渲染 |
| 代码高亮 | highlight.js | 代码片段着色 |
| CSS 方案 | Tailwind CSS | 原子类，快速开发 |
| 图标 | Lucide React | SVG 图标库 |

### 5.3 IPC 通道设计

```
搜索浮窗相关：
  search:links  (keyword)        → 返回匹配链接列表
  link:open     (url)            → 主进程用 shell.openExternal() 打开浏览器
  window:open-main               → 打开详情窗口

详情窗口相关：
  links:list    (filter?)        → 链接列表
  links:create  (title,url,tags) → 新建链接
  links:update  (id, data)       → 更新链接
  links:delete  (id)             → 删除链接
  notes:*                        → 笔记 CRUD（同上模式）
  snippets:*                     → 代码片段 CRUD
  clipboard:list (limit, offset) → 剪切板历史
  clipboard:copy (id)            → 复制到剪切板
  settings:get  (key)            → 读取设置
  settings:set  (key, value)     → 写入设置
```

## 6. 数据库设计

### 6.1 表清单

```
links             — 链接
notes             — 笔记
snippets          — 代码片段
clipboard_history — 剪切板历史
settings          — 系统设置
```

### 6.2 表结构

**links**

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER PK AUTOINCREMENT | 主键 |
| title | TEXT NOT NULL | 链接名称 |
| url | TEXT NOT NULL | 链接地址 |
| remark | TEXT | 备注说明 |
| tags | TEXT | 标签 JSON 数组 |
| favicon | TEXT | 网站图标 URL（可选） |
| open_count | INTEGER DEFAULT 0 | 打开次数 |
| created_at | TEXT DEFAULT (datetime('now')) | |
| updated_at | TEXT DEFAULT (datetime('now')) | |

**notes**

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER PK AUTOINCREMENT | |
| title | TEXT NOT NULL | |
| content | TEXT | Markdown 正文 |
| tags | TEXT | 标签 JSON 数组 |
| created_at | TEXT DEFAULT (datetime('now')) | |
| updated_at | TEXT DEFAULT (datetime('now')) | |

**snippets**

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER PK AUTOINCREMENT | |
| title | TEXT NOT NULL | |
| code | TEXT NOT NULL | 代码内容 |
| language | TEXT | 语言标识 |
| tags | TEXT | 标签 JSON 数组 |
| created_at | TEXT DEFAULT (datetime('now')) | |
| updated_at | TEXT DEFAULT (datetime('now')) | |

**clipboard_history**

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER PK AUTOINCREMENT | |
| content | TEXT NOT NULL | 文本内容 |
| content_hash | TEXT NOT NULL UNIQUE | SHA256 去重 |
| copied_at | TEXT DEFAULT (datetime('now')) | |

**settings**

| 字段 | 类型 | 说明 |
|------|------|------|
| key | TEXT PK | |
| value | TEXT | JSON |

### 6.3 索引

- `links.title`, `links.url` — 搜索浮窗模糊匹配
- `links.open_count` DESC — 按频率排序
- `notes.updated_at` DESC
- `snippets.language`, `snippets.updated_at` DESC
- `clipboard_history.copied_at` DESC, `clipboard_history.content_hash` UNIQUE
