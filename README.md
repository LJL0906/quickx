<h1 align="center">QuickX</h1>
<p align="center">Windows 桌面效率工具 · 随叫随到，用完即走</p>

---

## ✨ 功能

| 模块 | 说明 |
|------|------|
| 🔗 **链接管理** | 收藏网址，一键搜索打开，支持标签分类和备注 |
| 📝 **笔记** | Markdown 实时预览，左右分栏编辑，Ctrl+S 保存 |
| 💻 **代码片段** | 语法高亮，按语言分类，一键复制代码 |
| 📋 **剪切板** | 自动记录复制历史，点击回贴，实时刷新 |
| ⚙️ **设置** | 快捷键自定义、开机自启动、数据导入导出 |

---

## ⌨️ 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Alt + Q` | 唤起搜索浮窗 |
| `Alt + `` ` | 打开链接管理 |
| `Alt + 1` | 打开笔记 |
| `Alt + 2` | 打开代码片段 |
| `Alt + 3` | 打开剪切板 |
| `Alt + 4` | 打开设置 |

> 所有快捷键均可在设置页面自定义修改

---

## 📦 安装

1. 前往 [Releases](https://github.com/LJL0906/quickx/releases) 下载最新 `QuickX Setup x.x.x.exe`
2. 双击安装，选择安装目录
3. 安装完成后自动启动，托盘出现 QuickX 图标
4. 按 `Alt + Q` 开始使用

---

## 🚀 快速上手

- **搜索链接**：`Alt+Q` → 输入关键词 → 回车打开浏览器
- **记笔记**：`Alt+1` → 左侧新建 → 写 Markdown → `Ctrl+S` 保存
- **存代码**：`Alt+2` → 新建片段 → 粘贴代码 → 自动高亮
- **查剪切板**：`Alt+3` → 点击任意记录回贴
- **改快捷键**：`Alt+4` → 点击快捷键 → 按下新组合键

---

## 🔧 特性

- 开机自启动，常驻系统托盘
- 全局快捷键，任何窗口下都能响应
- 数据本地存储（SQLite），无需联网
- 支持数据导出/导入备份
- 自动检测 GitHub 更新，一键升级
